# 项目资料管理平台使用手册

## 1. 系统概述

项目资料管理平台是一个专为项目全生命周期文档管理设计的系统，提供文档上传、管理、审批、验收等功能，帮助项目团队高效管理项目文档，确保项目资料的完整性和可追溯性。

### 1.1 主要功能
- 项目创建与管理
- 文档上传与分类
- 文档审批与验收
- 权限管理与控制
- 数据统计与分析
- 消息通知系统

### 1.2 系统架构
- 前端：HTML5 + CSS3 + JavaScript
- 后端：Python Flask
- 数据库：SQLite
- 部署：支持 Windows、Linux、macOS

## 2. 安装指南

### 2.1 环境要求
- Python 3.8 或更高版本
- 操作系统：Windows 7+、Linux、macOS
- 浏览器：Chrome、Firefox、Edge、Safari

### 2.2 安装步骤

#### 2.2.1 Windows 系统
1. 下载项目源码
2. 解压到本地目录
3. 运行 `install.bat` 脚本安装依赖
4. 运行 `start.bat` 启动服务器
5. 打开浏览器访问 `http://localhost:5000`

#### 2.2.2 Linux/macOS 系统
1. 下载项目源码
2. 解压到本地目录
3. 运行 `chmod +x Daemon.sh` 赋予执行权限
4. 运行 `./Daemon.sh install` 安装依赖
5. 运行 `./Daemon.sh start` 启动服务器
6. 打开浏览器访问 `http://localhost:5000`

- 测试账户：test001 / admin123

## 4. 系统功能使用

### 4.1 项目管理

#### 4.1.1 创建项目
1. 点击左侧导航栏"项目管理"
2. 点击"创建项目"按钮
3. 填写项目基本信息
4. 点击"保存"按钮

#### 4.1.2 项目列表
- 查看所有项目列表
- 按状态筛选项目
- 搜索项目
- 点击项目进入详情页

#### 4.1.3 项目详情
- 查看项目基本信息
- 管理项目周期
- 查看项目文档
- 查看项目审批状态

### 4.2 文档管理

#### 4.2.1 上传文档
1. 进入项目详情页
2. 选择项目周期
3. 点击"上传文档"按钮
4. 选择文件并上传
5. 填写文档信息
6. 点击"保存"按钮

#### 4.2.2 文档分类
- 按文档类型分类
- 按上传时间排序
- 按状态筛选

#### 4.2.3 文档操作
- 预览文档
- 下载文档
- 编辑文档信息
- 删除文档
- 归档文档

### 4.3 审批流程

#### 4.3.1 提交审批
1. 上传文档后，点击"提交审批"
2. 选择审批人
3. 填写审批说明
4. 点击"提交"按钮

#### 4.3.2 审批操作
1. 审批人登录系统
2. 点击"审批管理"
3. 查看待审批文档
4. 点击"审批"按钮
5. 选择"通过"或"拒绝"
6. 填写审批意见
7. 点击"提交"按钮

## 5. 权限管理

### 5.1 角色权限

| 角色 | 权限说明 |
|------|----------|
| admin | 系统管理员，拥有所有权限 |
| pmo | 项目管理办公室，拥有项目管理和审批权限 |
| project_admin | 项目经理，拥有本单位项目管理权限 |
| contractor | 承建单位用户，拥有本单位项目文档管理权限 |

### 5.2 权限控制
- 管理员：可以访问所有项目和功能
- PMO：可以访问所有项目，具有审批权限
- 项目经理：只能访问本单位的项目
- 承建单位：只能访问本单位的项目和文档

## 7. 常见问题

### 7.1 登录问题
- **问题**：无法登录系统
  **解决方案**：检查用户名和密码是否正确，联系管理员重置密码

- **问题**：账户被锁定
  **解决方案**：联系管理员解锁账户

### 7.2 上传问题
- **问题**：无法上传文档
  **解决方案**：检查文件大小是否超过限制，检查网络连接

- **问题**：上传速度慢
  **解决方案**：检查网络连接，尝试上传较小的文件

### 7.3 审批问题
- **问题**：审批流程卡住
  **解决方案**：联系审批人，检查审批状态

- **问题**：审批通知未收到
  **解决方案**：检查消息通知设置，联系系统管理员

### 7.4 系统问题
- **问题**：系统响应慢
  **解决方案**：检查服务器状态，联系系统管理员

- **问题**：系统崩溃
  **解决方案**：刷新页面，联系系统管理员
