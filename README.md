# 项目文档管理中心 📁

一个功能强大的项目全生命周期文档管理系统，支持文档收集、版本管理、自动签字/盖章识别和缺失文档统计。

## ✨ 核心功能

### 🎯 文档收集与管理
- 按项目生命周期竖向展示文档结构
- 支持多个文档目录，每个目录可上传多个文档
- 实时AJAX文件上传，无需页面刷新
- 支持拖拽上传
- 文档版本管理和替换

### 🔍 智能识别
- **自动签字检测**：使用边缘检测识别文档是否已签字
- **自动盖章识别**：通过颜色分析识别红章或蓝章印章
- 识别置信度显示

### 📊 缺失文档统计
- 实时统计各周期缺失的文档
- 生成详细的完成率报告
- 按周期展示完成进度

### 📚 文档元数据
- 记录文档上传时间
- 文档日期、签字日期、签字人信息
- 文件大小和类型

### 📈 报告生成
- 生成项目文档管理报告
- 可视化完成率进度条
- 导出为HTML文件

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用（开发模式）

```bash
python main.py
```

应用会在 `http://localhost:5000` 启动

### 3. 使用应用

1. 打开浏览器访问应用
2. 点击"加载项目配置"上传需求清单Excel文件
3. 选择左侧的项目周期
4. 在右侧拖拽或点击上传文档
5. 填写文档元数据（日期、签字人等）
6. 生成报告查看完成情况

## 🔧 API路由

### 项目管理
- `POST /api/project/load` - 加载项目配置

### 文档操作
- `POST /api/documents/upload` - 上传文档
- `GET /api/documents/list` - 获取文档列表
- `DELETE /api/documents/<doc_id>` - 删除文档

### 报告生成
- `POST /api/report` - 生成文档管理报告

### 页面
- `GET /` - 主页面

## 🔬 识别算法说明

### 签字识别
1. 读取图像并转换为灰度
2. 使用CLAHE增强对比度
3. Canny边缘检测
4. 计算边缘像素比例
5. 阈值判定（>0.01为有签字）

### 盖章识别
1. 转换为HSV颜色空间
2. 检测红色范围 (H: 0-10, 170-180)
3. 检测蓝色范围 (H: 100-130)
4. 计算有色像素比例
5. 阈值判定（>0.002为有盖章）

## 🐛 常见问题

### Q: 上传的文件在哪里？
A: 默认保存在 `uploads/` 目录下，按项目周期和文档名称分类。

### Q: 可以删除已上传的文档吗？
A: 可以。在文档管理弹窗中点击"删除"按钮即可删除任何文档。

### Q: 签字和盖章识别准确度如何？
A: 对于清晰的图像，准确度约90%+。识别结果会显示置信度供参考。

### Q: 支持哪些文件格式？
A: 支持 PDF、Word、Excel、PNG、JPG、JPEG、TIFF 等常见文档和图片格式。

### Q: 能否批量上传？
A: 目前支持单个文件上传。可以依次打开多个文档窗口进行快速上传。
